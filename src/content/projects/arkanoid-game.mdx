## Project Overview

A classic Arkanoid-style breakout game developed in Python using the Pygame library. The player controls a paddle to bounce a ball and break bricks — a timeless arcade concept that serves as an excellent playground for learning game development fundamentals.

## Game Features

- **Classic Gameplay:** Paddle-and-ball mechanics faithful to the original Arkanoid
- **Collision Detection:** Precise ball-brick and ball-paddle collision handling
- **Score Management:** Track and display player score in real-time
- **Game Loop:** Smooth 60fps rendering with proper frame timing
- **Player Controls:** Responsive keyboard controls for paddle movement

## Technical Approach

The game is built using object-oriented design principles, with separate classes for the paddle, ball, bricks, and game manager. Pygame handles rendering, input, and the game loop.

Key technical challenges included:
- Implementing accurate collision detection at the edges and corners of bricks
- Managing game state transitions (start, play, game over)
- Keeping the physics consistent across different frame rates

## What I Learned

This project was my introduction to game development concepts — game loops, collision detection, sprite management, and real-time rendering. It also reinforced object-oriented programming patterns in Python and showed me how fundamental CS concepts like vectors and state machines apply to game development.
