## Project Overview

A clean, browser-based weather application built entirely with vanilla TypeScript — no frameworks, no build tools, just pure TypeScript and the WeatherAPI.com API.

## Features

- **City Search:** Look up weather for any city worldwide
- **Current Weather:** Real-time temperature, conditions, humidity, and wind data
- **5-Day Forecast:** Extended forecast with daily highs, lows, and conditions
- **Favorites:** Save your frequently checked locations for quick access
- **Notification Alerts:** Set up weather alerts for specific conditions

## Technical Approach

The goal was to build a fully functional weather app without relying on any frameworks — proving that vanilla TypeScript is more than capable for interactive web applications. The app fetches data from the WeatherAPI.com REST API and renders everything with DOM manipulation.

Key decisions included:
- Using TypeScript for type safety across the entire codebase
- Implementing local storage for persisting favorite locations
- Designing a responsive layout with pure CSS

## What I Learned

Building without a framework forced me to deeply understand how the DOM works, how to manage state manually, and how to structure a TypeScript project cleanly. It was a great exercise in fundamentals.
